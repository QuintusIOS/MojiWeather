# MojiWeather
支持国内外各城市天气情况，提供一周天气预报。可查询空气指数能见度，紫外线，日出日落时间预报，同时支持手机定位获取天气。外出旅行，出门上班必备APP。
##### 使用技术
-	JSON数据格式解析、数据存储方式处理和大批量的数据库存储管理。
-	利用Core Location提供经纬度实现定位功能。
-	使用UIWebView加载网页数据,浏览网页信息。
-	借助工具Reveal进行UI调试。
